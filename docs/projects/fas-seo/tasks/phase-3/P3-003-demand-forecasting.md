Build time-series demand forecasting using 12+ months of Medusa transaction data.

Forecast dimensions:
- Product category (pulleys, packages, wheels, etc.)
- Vehicle platform (S550, TRX, F-150, etc.)
- Season (holiday, spring build season, etc.)

Requires significant historical data — cannot start until Phase 1 instrumentation has been running for 12+ months.
